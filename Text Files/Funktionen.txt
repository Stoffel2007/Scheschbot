Kommando-Funktionen:
    - help:
        gibt einen Überblick über verfügbare Aktionen
    - cook:
        verschiedene Gerichte stehen zur Auswahl, wenn man Pech hat, isst der Scheschbot einem alles weg
    - pr0:
        ein Bild wird aus pr0gramm geholt, entweder per Zufall, Suchbegriff, ID oder das neueste
    - pr0gif:
        ein GIF wird aus pr0gramm geholt, entweder per Zufall, Suchbegriff, ID oder das neueste
    - pr0vid:
        ein Video wird aus pr0gramm geholt, entweder per Zufall, Suchbegriff, ID oder das neueste
    - Aggronyme:
        Bot beschreibt einen Namen anhand der einzelnen Buchstaben
    - translate

Inline-Funktionen:
    - randomize:
        @Scheschbot <beliebiger Text> => <beigeliebr Txet>

andere Funktionen:
    - answer:
        Bot reagiert auf bestimmte Wörter und Sätze (teilweise mit Vorbedingungen)
        Idee: auf meherere Fragen in einer Nachricht (durch Satzzeichen getrennt) separat antworten
    - randomschesch:
        Bot postet manchmal irgendwelche zufälligen Nachrichten
        Idee: Bot postet zufälligen Sticker aus dem Package Bach-Grötsch
    - mood:
        Wenn jemand bestimmte Wörter verwendet (Ausdrücke, Nettigkeiten), mag er diese Person mehr oder weniger.
        Diese Skala beeinflusst das Verhalten verschiedener Funktionen (Nudeln wegessen usw.)

    - Stupsfunktion (inline)
